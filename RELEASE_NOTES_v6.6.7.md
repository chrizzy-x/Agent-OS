# AgentOS V6.6.7

Super AgentOS becomes the primary product surface.

## Highlights

- Workspace Context Engine for user, workspace, memory, Vault metadata, tasks, and capability graph context.
- Capability Graph for apps, skills, workflows, subagents, MCP tools, projects, Library assets, and system capabilities.
- Durable task records, task steps, approval records, and Super AgentOS audit logs.
- Confirmation-aware capability execution with no fake success for unavailable or unconfigured capabilities.
- Task Center at `/tasks` for running, queued, approval, completed, failed, and cancelled work.
- Studio streaming now creates persisted task records and answers workspace capability questions from installed workspace state.
