# AgentOS Navigation

Top-level order:

Home, Studio, Projects, Workflows, Library, App Store, Developer, FFP, Settings.

Apps, Skills, Subagents, Memory, Vault, and Connectors are Library sections, not primary destinations. FFP is visible as disabled Coming Soon. No module is placed behind a More menu.

The left sidebar is 280px expanded and 72px collapsed. Collapse state is stored in `localStorage`. Below 768px the left and right rails become accessible drawers.

The sidebar exposes workspace selection above navigation. Mobile bottom navigation is limited to Home, Studio, Library, Workflows, and Settings.
